# Ynera 3

Síntesis de las dos direcciones visuales de Ynera: el jardín orgánico de la primera versión y la claridad de estudio tecnológico de la segunda.

## Ejecutar localmente

```bash
npm install
npm start
```

El sitio queda disponible en `http://localhost:3000`.

## Sistema visual

- Fondo bosque oscuro
- Verde vivo para datos, crecimiento y continuidad
- Violeta para IA, intervención y llamados a la acción
- Tipografía Fraunces + Manrope + IBM Plex Mono

El sitio es HTML, CSS y JavaScript sin framework, servido con Express.
